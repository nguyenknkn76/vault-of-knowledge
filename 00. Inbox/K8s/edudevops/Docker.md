## How to use Docker?


### Install Docker

```sh
# install-docker.sh

```

How to run sh file? 
```bash
bash install-docker.sh

sh install-docker.sh

./ install-docker.sh
```

Find images in Docker Hub and pull it
```bash
# sample pull image from Docker Hub
docker pull ubuntu:22.04
docker images
docker run --name ubuntu -it ubuntu:22.04
```
